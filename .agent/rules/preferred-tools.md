# Preferred Tools

Use modern CLI tools for better efficiency:

| Task                  | Use            | Don't Use             |
| --------------------- | -------------- | --------------------- |
| Find files            | `fd`           | `find`, `ls -R`       |
| Search text           | `rg` (ripgrep) | `grep`, `ag`          |
| Interactive selection | `fzf`          | Manual filtering      |
| Process JSON          | `jq`           | `python -m json.tool` |
